## 第二次迭代：苹果风 → 量子暗黑科技风

### 用户反馈
- 科技感不足，界面太"土"
- 部分 UI 溢出文本框

### 修改内容
1. **全局暗色模式**：背景从 #F5F5F7 → 深海军蓝渐变（#080C18 → #0E1428）
2. **配色方案**：Quantum Computing 风格——量子青 #00E5FF + 干涉紫 #7C3AED + 深海军蓝底
3. **字体升级**：标题用 Orbitron Bold（科技感）+ 正文 Geist + 代码 JetBrains Mono
4. **按钮升级**：Cyan→Purple 渐变 + 发光阴影
5. **卡片升级**：暗色玻璃态（#0D111C）+ 微妙边框 + 青蓝辉光阴影
6. **溢出修复**：所有卡片启用 clipsContent，消息输出面板加高至 280px，内容区加高至 180px
7. **统计卡片**：全部改为暗色 + 发光边框
8. **Agent 管道**：三步骤暗色卡片 + 各自颜色辉光（绿/青/暗）
9. **导航栏**：全部导航图标和标签颜色适配暗色主题

### 设计系统
- 背景: #080C18 → #0E1428 渐变
- 卡片: #0D111C 带 rgba(255,255,255,0.06) 边框
- 强调色: #00E5FF（主）、#7C3AED（辅）
- 文字层级: #E2E8F0（标题）→ #8892B0（正文）→ #64748B（辅助）
- 辉光效果: rgba(0, 0.55, 0.9, 0.06) 蓝色扩散阴影
